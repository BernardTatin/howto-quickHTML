This project contains a set of demonstration files (step1, step2, ... directories) and a mini site which explains theses steps. You had to install at least the mini site and the steps.

installing the site

You must install an HTTP server (Apache, NGinx, lighttpd...) because the site use Ajax queries. some browsers need special configuration to work with Ajax queries on the hard disk.

The next step is really easy, download the ZIP archive, decompress it in a place where a HTTP server can see it.

Note: with last versions of Firefox, you can access to the site directly (File -> Open... menu). Ajax will work, but I am not sure all the Javascript I use works.

playing with the code

You must install Less, a CSS preprocessor because I generate CSS with it.

To explore, modify... source code, you can use :

  • Netbeans,
  • Eclipse,
  • Bracketts,
  • a good code editor...

I use Bracketts.

Note: each time you modify a less file, don't forget to apply less on the main.css!
